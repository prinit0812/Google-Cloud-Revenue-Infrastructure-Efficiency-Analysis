# Google-Cloud-Revenue-Infrastructure-Efficiency-Analysis
Built an end-to-end cloud finance analytics solution using SQL, Python, Excel, and Power BI to analyze revenue performance, infrastructure costs, profitability, and forecasting scenarios across Google Cloud products and regions.

**Overview**

This project analyzes Google Cloud revenue performance, infrastructure utilization, and operational efficiency across products, regions, and customer segments. The objective is to identify revenue growth drivers, optimize infrastructure spending, improve forecasting accuracy, and support strategic decision-making through data-driven insights.

The analysis simulates a Cloud Finance and FP&A workflow similar to what financial analysts at large technology companies perform to evaluate business performance, capacity planning, and profitability.
